Custom Instructions — Your Profile

This is the single most important setup step. Claude reads your profile on every conversation and tailors responses to *you*.

Start here → Fill in the fields below, then click "Generate" to build a profile block. Paste it into Settings → Profile in Claude.

Project Builder

Projects let Claude carry context across conversations. Pick a starter, edit the system prompt, paste it into a new Claude project.

Start here → Set up at least one project today. Content/Writing is the easiest to start — you'll feel the difference immediately.

◇ Client Work

◇ Admin & Ops

Connectors — Apps to Connect

Connectors let Claude actually do things in your tools. Turn them on one at a time so you understand what you're granting.

Start here → Google Calendar + Gmail first. They unlock the most useful daily wins (calendar audits, inbox triage, follow-ups).

Skills Planner

Skills are reusable instructions Claude triggers automatically when you mention a topic. Like having pre-trained shortcuts for your most-repeated work.

Start here → Build one skill this week. Writing Voice is the highest-leverage place to begin.

✦ Writing Voice

Triggers when you ask Claude to draft, write, edit, or rewrite anything for publication.

✦ Content Repurposer

Triggers when you give Claude a long piece and ask for it broken into formats.

✦ Client Email Drafter

Triggers when you need to write or reply to a client email.
